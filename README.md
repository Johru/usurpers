# Usurper's Market Builder

Tableau builder for Usurpers Market board game. Users can brainstorm ideas by quickly assembling 7 card lineups, save them for later and edit past ideas. Since data is fairly simple and for individual use only, there is no need for a backend, data is saved locally. Json backup option will be added later.

## Live Demo

🔗 https://usurpers.vercel.app/contact

## Tech Stack

- TypeScript
- React / NextJS
- CSS / Tailwind
